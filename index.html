Welcome to PandoraBox

PandoraBox is a new generation content publishing/gathering platform,we give a self definition to it--"Rich Media FM". The minimum content unit, which called "box" in the system, is consists of a set of musics, a set of pictures,a title and a text which act as the main description of the editable unit. We mixed such kinds of medias into a single page with a user-defined layout, in that case users can make any content units with the draw they want. Pandora provided functionality to display the content unit in specific rules, the users can look up and play the contents (box) they interested in.
